Tutoriel / Обучение

Salut! Je suis Bulgarche, votre oiseau guide! / Здравей! Аз съм Българче, вашата птица водач!

Je vais vous montrer comment utiliser cette application! / Ще ви покажа как да използвате това приложение!

1. Accueil / Начало

Sur la page d'accueil, vous verrez vos statistiques de progression.

На началната страница ще видите вашата статистика за напредък.

2. Leçons / Уроци

Suivez le chemin coloré pour progresser dans les leçons. Marquez chaque leçon comme lue.

Следвайте цветния път, за да напредвате в уроците. Маркирайте всеки урок като прочетен.

3. Tests / Тестове

Testez vos connaissances avec des quiz basés sur les leçons que vous avez étudiées.

Тествайте знанията си с тестове, базирани на уроците, които сте изучавали.

4. Dictionnaire / Речник

Recherchez des mots et écoutez leur prononciation avec les fichiers audio.

Търсете думи и слушайте тяхното произношение с аудио файловете.

5. Jeux / Игри

Apprenez en jouant! Les jeux vous aident à mémoriser le vocabulaire.

Учете се играейки! Игрите ви помагат да запомните лексика.
